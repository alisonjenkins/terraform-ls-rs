## 0.2.3

- Initial release of the dedicated VS Code extension for the `tfls`
  (terraform-ls-rs) language server.
- Auto-downloads the matching `tfls` binary for Linux x64, macOS arm64, and
  Windows x64, with checksum verification and a `serverPath` override.
- Live `formatStyle` toggle (minimal ↔ opinionated) and full `terraform-ls-rs.*`
  configuration surface.
